Site Content

My name is Brendan Ferreri-Hanberry and I started this site as a project for a web development course in graduate school. Currently this site includes HTML5, Javascript and some CSS features such as flexboxes and the nav bar above. It also demonstrates mobile-first responsive design, while another page features a responsive table implemented with jQuery. The Home Town page features an interactive map implemented with Javascript, and on another page I implement a Bootstrap element called Font Awesome. The photography on the site is my own, done with a Nikon COOLPIX L830, and you can find more of my pictures on Flickr.
